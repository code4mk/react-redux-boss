# react-redux-boss (RRB)
RRB is a mini React Redux boilerplate .

<p align="center" ><img src="./src/assets/rrb1.png"></p>

# Main Feature:

 - React
 - React Router Dom
 - Redux (coming soon)
 - Axios
 - Bulma

# How use RRB

 * `Clone RRB project`
 * `yarn install` or `npm install`
 * `npm run dev`

# Our React Atom Packages:
 - `apm install atom-react`
 - `apm install atom-axios`
